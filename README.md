# LibStudioElttob
A mirror of Studio Elttob's in-house custom packages for Roblox development.
**Please note that this repository is a read-only mirror, and does not accept code contributions.**

## Notes

Some libraries contained non-code assets which have not been licensed for source
availability. These assets have been removed; this repository does not grant
permission to use those assets.

## Dependencies

LibStudioElttob depends on [LibOpen](https://github.com/Elttob/LibOpen). Packages in LibStudioElttob may depend on each other; they should be placed as siblings in the data model.
